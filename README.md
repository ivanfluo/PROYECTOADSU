# PROYECTOADSU

Sistema de Votación Electrónica desarrollado para el curso de Análisis de Sistemas 1.

## Descripción

El proyecto consiste en el desarrollo de una plataforma de votación electrónica donde los ciudadanos podrán registrarse utilizando su DPI, iniciar sesión y emitir su voto de manera digital.

El sistema valida que cada ciudadano únicamente pueda votar una vez por elección, permitiendo mantener integridad en los resultados electorales.

Además, el sistema cuenta con:
- Gestión de elecciones
- Gestión de candidatos
- Resultados electorales
- Bitácoras de acciones
- Validación de voto único

---

## Objetivos del Proyecto

- Automatizar el proceso de votación electrónica.
- Garantizar que un ciudadano vote únicamente una vez por elección.
- Facilitar la administración de elecciones y candidatos.
- Permitir auditoría mediante bitácoras del sistema.
- Aplicar metodología Scrum durante el desarrollo.
